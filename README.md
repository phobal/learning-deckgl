# Learning-deck.gl

## Prepare
```shell
git clone https://github.com/vite-react-ts-antd-starter.git
cd vite-react-ts-antd-starter
pnpm i # If not install pnpm, please run: npm install -g pnpm
```

## Usage

### Development

```shell
pnpm dev
```
and click it http://localhost:5173
### Build

```shell
pnpm build
```
